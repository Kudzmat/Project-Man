# Project-Man
A project management system for structural engineering firm

A Java program which uses OOP to create an app that can be used to keep track of the many projects on which they work.
This program uses .txt files as databses to store information 

The company stores the following information for each project that they work on:
● Project number.
● Project name.
● What type of building is being designed? E.g. House, apartment block or
store, etc.
● The physical address for the project.
● ERF number.
● The total fee being charged for the project.
● The total amount paid to date.
● Deadline for the project.
● The name, telephone number, email address and physical address of the
architect for the project.
● The name, telephone number, email address and physical address of the
contractor for the project.
● The name, telephone number, email address and physical address of the
customer for the project.
